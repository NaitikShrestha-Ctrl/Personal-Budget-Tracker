# Personal Budget Tracker
A modern desktop budget management application built with Python Tkinter that helps users track expenses, manage budgets, analyze spending habits, and export reports inside a clean dark-themed interface.

Description
Personal Budget Tracker is a Python-based GUI application designed for users who want to manage their personal finances easily. It allows users to record expenses, monitor budgets, view analytics, and organize spending records in one place.

Features
User Authentication
    Login and Sign Up system
    Secure password storage using SHA-256 hashing

Expense Management
    Add new expenses with:
        Amount
        Category
        Description
        Date
        Notes
    Delete expenses
    View recent transactions

Dashboard Overview
    Monthly spending summary
    Total spending
    Average expense per entry
    Remaining monthly budget
    Recent transaction history

Analytics
    Spending by category
    Monthly trend charts
    Category share visualization
    Budget performance indicators

Transaction History
    Search expenses
    Filter by category
    Sort by newest, oldest, highest, or lowest

Budget Management
Category Budget
    Set spending limits for each category

Monthly Budget
    Set monthly total spending goals

Reports and Export
    Export all expenses to CSV format

User Interface
    Clean dark theme design
    Sidebar navigation
    Responsive layout

Built With
    Python 3
    Tkinter (GUI Library)
    JSON (Local Database Storage)
    CSV
    hashlib

Data Storage
All data is stored locally in:
pbt_dark.json

Stored data includes:
    Users
    Expenses
    Category Budgets
    Monthly Budgets

How to Run
    Install Python 3 from python.org
    Run the application:
    python PersonalBudgetTracker.py

Main Modules
